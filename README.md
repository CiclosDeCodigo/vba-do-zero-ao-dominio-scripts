## Sobre este repositório

Este repositório contém os códigos, exemplos e automações apresentados no livro **VBA — Do Zero ao Domínio**, com foco em:

- Automação de tarefas no Excel  
- Desenvolvimento de macros em VBA  
- Manipulação de dados e objetos do Excel  
- Organização e arquitetura de código  
- Performance, tratamento de erros e boas práticas  
- Uso de VBA em ambientes corporativos  
- Integração com arquivos, sistemas e outras aplicações  
- Uso de VBA em conjunto com Inteligência Artificial  

Todos os códigos seguem boas práticas e são documentados para uso seguro em ambientes reais.
